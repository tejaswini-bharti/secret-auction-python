# 🐍 Secret Auction Program

## 📌 Project Overview
This is a simple Python project that simulates a **Secret Auction System**.  
Multiple users can place bids, and the program determines the highest bidder automatically.

---

## 🚀 Features
- Accepts multiple bidders
- Keeps bids hidden from other users
- Determines the highest bidder
- Simple and easy-to-use command line interface

---

## 🛠️ Technologies Used
- Python
- Dictionaries
- Functions
- Loops & Conditional Statements
- `os` module

---

## ▶️ How to Run the Program

1. Clone this repository or download the file  
2. Open the project folder  
3. Run the Python file:
